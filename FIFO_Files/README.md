**Цель:** Разработать систему учета товаров с автоматическим списанием партий по срокам годности.

**Основные требования:**
* Учет товаров в разрезе сроков годности (партионный учет).
* Автоматическое списание по методу FIFO (в первую очередь списываются товары, срок годности которых истекает быстрее).
* Контроль отрицательных остатков в момент проведения.
* Возможность корректной работы при вводе документов "задним числом".
* Отчет на СКД по остаткам с сортировкой по срокам годности.

## 🛠 Технические особенности реализации
В решении применены стандарты разработки на платформе 1С:Предприятие 8.3:

1. **Оптимизация запроса:** Контроль остатков и получение данных для списания реализованы **одним запросом** с использованием временных таблиц и итогов (`ИТОГИ ПО Товар`).
2. **Управляемые блокировки:** Исключена ситуация коллизий при одновременной работе пользователей. Блокируются только те позиции номенклатуры, которые присутствуют в документе.
3. **Работа в прошлом периоде:** Для получения корректных остатков на момент времени документа (а не на текущую дату) использован объект `Граница` с видом `Исключая`.
4. **Чистый код:** Логика вынесена в модуль объекта документа. Использованы адекватные типы данных (точность 3 знака для весового товара).
