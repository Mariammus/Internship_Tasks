# Задача №6. Скоропортящиеся продукты. Вариант 2

<details>
<summary>🚩 Условие задачи (нажми, чтобы развернуть)</summary>

**Цель:** Разработать систему учета товаров в разрезе сроков годности.

**Основные требования:**
*   **Учет товаров:** Ведется партионный учет по срокам годности (для каждого товара свой срок). 
*   **Точность:** Учет веса до граммов (3 знака после запятой).
*   **Цены:** Регистрация и хранение истории продажных цен. Автоматическая подстановка цены в документ.
*   **Продажа:** Выбор конкретной партии (срока годности) вручную. Ограничение списка выбора только теми партиями, что есть в наличии.
*   **Контроль:** Запрет продажи "в минус". Корректная работа при вводе документов "задним числом".
*   **Отчетность:** Построение отчета по остаткам партий товаров.
</details>

### 🛠 Технические особенности реализации
В решении применены стандарты разработки на платформе 1С:Предприятие 8.3:

1. **Оптимизация запроса:** Контроль остатков реализован пакетным запросом с использованием временных таблиц и индексации.
2. **Управляемые блокировки:** Исключена ситуация коллизий. Блокируются только те позиции номенклатуры и сроки годности, которые присутствуют в документе.
3. **Работа в прошлом периоде:** Для получения корректных остатков на момент времени документа использован объект `Граница` с видом `Включая`.
4. **Чистый код:** Логика проведения вынесена в модуль объекта. Реализован динамический фильтр сроков годности в форме с учетом текущих остатков и данных в ТЧ.
